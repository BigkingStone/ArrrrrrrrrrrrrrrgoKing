## Commit Convention
#### commit message form
```bash
[{commit-type}] #{issue-number} {commit-message}
```

#### commit type
* `[Add]`: 새로운 파일 추가(ex. 풀이한 새로운 코드 파일)
* `[Chore]`: 코드 수정(ex. 이미 풀이한 코드에서 로직 수정)
* `[Docs]`: 문서 파일 추가(ex. Readme 파일 or 풀이 과정을 담은 문서)
* `[Delete]`: 파일 삭제
